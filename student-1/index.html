Chart Description

This Donut Chart was created using d3(Data-Driven Documents). The chart shows the operating profit for all regions within the dataset.

the donut chart is created using three columns from the Adidas Sales dataset: Region, Operating_Profit, and Year. The operating profit for each region is summed up, and the dataset is filtered based on the selected year, rendering the operating profit for each region for that specific year. This approach provides a focused visualization of how operating profits are distributed across regions over time, allowing users to explore the data interactively by selecting different years.
